## How to Run

### VS Code Command

Open terminal in VS Code and run:

```bash
mvn javafx:run
```

Or use the task:
- Press `Ctrl+Shift+P`
- Type: "Tasks: Run Task"
- Select: "Run Library Management System"

## Visual Features

### Color Scheme
- **Background**: #0a1929 (Deep Navy)
- **Secondary BG**: #1a2332 (Charcoal)
- **Primary Accent**: #00d4ff (Electric Blue)
- **Secondary Accent**: #0ea5e9 (Bright Blue)
- **Text**: #b0c4de (Light Blue-Gray)
- **Success**: #27ae60 (Green)
- **Danger**: #e74c3c (Red)

### Effects
- **Box Shadow Glow**: `dropshadow(gaussian, rgba(0, 212, 255, 0.3), 20, 0, 0, 0)`
- **Border Glow**: `rgba(0, 212, 255, 0.3)` to `rgba(0, 212, 255, 0.6)` on hover
- **Glass Effect**: `rgba(26, 35, 50, 0.7)` with blur
- **Hover Scale**: 1.02x to 1.05x
